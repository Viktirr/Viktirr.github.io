Concept Tycoon is a Roblox Tycoon game built mostly from the ground up, almost no third-party tools were used to create this game.

The game was built in collaboration with the Kilovat group, I was tasked to create most of the systems for the game.

Mostly worked on the following:

  • Tycoon Management System
    • Creates tycoons upon game startup using one base tycoon
    • Tycoon cleanup after a player leaves
    • Modifies tycoons according to their purpose
  • Tycoon Functionality
    • Upgrades
    • Purchases
    • Purchase animations
    • Game currency
    • Tycoon ownership
    • Item collectors
  • Player Spawning
    • Randomised player spawning
    • Spawning inside of owned tycoon
    • Spawning players to where a tycoon is likely to be unclaimed

Built from March 2023 to May 2023.
